### Creating & Managing Keys

<Steps>
  <Step title="Generate New Key">
    Click "Create New Key" and specify:
    - Environment (dev/staging/production)
    - Permissions and restrictions
    - Expiration settings
    - Usage limits
  </Step>
  <Step title="Configure Security">
    Set up security parameters:
    - IP address restrictions
    - Domain whitelisting
    - Rate limiting rules
    - Access permissions
  </Step>
  <Step title="Copy & Secure">
    **Critical**: Copy the key immediately (shown only once)
    - Store in secure password manager
    - Update application configuration
    - Test API connectivity
  </Step>
  <Step title="Monitor Usage">
    Track key performance:
    - Request volume and patterns
    - Error rates and types
    - Geographic usage
    - Security alerts
  </Step>
</Steps>
