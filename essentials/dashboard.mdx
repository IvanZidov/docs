## Dashboard Overview

The VizAPI.ai dashboard is your central hub for managing templates, extractions, API keys, and account settings. This guide will help you navigate and make the most of every feature.

## Dashboard Layout

### Navigation Sidebar

The left sidebar provides quick access to all major sections:

- **Dashboard**: Overview and quick actions
- **Templates**: Browse and manage your templates
- **API Keys**: Manage authentication keys
- **Usage**: Monitor credits and billing
- **Settings**: Account and preferences

### Main Content Area

The main area displays context-specific information and tools based on your selected section.

## Dashboard Home

### Quick Start Section

The hero section at the top provides immediate access to key actions:

#### Templates Button
- Browse our curated library of pre-built templates
- See expert-crafted templates for common use cases
- One-click template deployment

#### New Extraction Button
- Launch the custom template builder
- Create templates tailored to your specific needs
- Access the AI-powered field suggestion engine

#### API Integration Panel
Shows code snippets for popular programming languages:
- **Python**: `requests.post(api_url, ...)`
- **JavaScript**: `fetch(apiUrl, { ... })`
- **Go**: `http.Post(apiURL, ...)`
- **PHP**: `curl_exec($ch);`
- **cURL**: `curl -X POST ...`

Each snippet provides a starting point for integration in your preferred language.

### Dashboard Metrics

Four key performance indicators give you instant insights:

#### Total Extractions
- Count of all extraction templates you've created
- Includes both active and inactive templates
- Click to view detailed breakdown

#### Total API Calls
- Cumulative count of all API requests across all templates
- Includes successful and failed requests
- Updates in real-time

#### Active This Week
- Templates that have been used in the past 7 days
- Helps identify your most valuable templates
- Quick filter to see recent activity

#### Most Used Template
- Your highest-traffic template by API call count
- Helps identify successful patterns
- Click to view template details

## My Extractions Table

### Column Details

#### Name
- Template display name
- Click to edit template
- Shows status indicators (active/inactive)

#### Created
- Template creation date
- Formatted as "MMM DD, YYYY"
- Sortable column

#### Last Used
- Timestamp of most recent API call
- Shows "Never" if unused
- Smart relative formatting:
  - "Just now" (< 1 minute)
  - "X minutes ago" (< 1 hour)
  - "X hours ago" (< 1 day)
  - "X days ago" (< 1 week)
  - Actual date (> 1 week)

#### Usage Count
- Total number of API calls for this template
- Badge styling for high-usage templates
- Click to view detailed analytics

#### Actions Menu
Each template has a three-dot menu with options:
- **Edit**: Modify template fields and settings
- **Clone**: Create a copy for testing variations
- **API Integration**: View code snippets and endpoint details
- **Analytics**: Detailed usage statistics
- **Delete**: Remove template (with confirmation)

### Batch Operations

Select multiple templates to perform bulk actions:
- **Bulk Delete**: Remove multiple templates at once
- **Export**: Download template configurations
- **Archive**: Move templates to archived state

## API Key Management

### Main API Key Section

Your primary API key provides full account access:

#### Key Display
- **Masked Key**: Shows first 8 characters + asterisks
- **Copy Button**: One-click copying to clipboard
- **Regenerate**: Create new key (invalidates old one)
- **Created Date**: When the key was generated

#### Security Features
- **Last Used**: Timestamp of most recent API call
- **Usage Location**: Geographic region of last use
- **Rate Limits**: Current limits and usage
- **Permissions**: Full account access indicator

### Additional API Keys

Create specialized keys for different use cases:

#### Key Creation
1. Click "Create New Key"
2. Add descriptive name (e.g., "Production App")
3. Optional description for context
4. Select permissions (full access or template-specific)
5. Copy the generated key immediately

#### Key Management
Each additional key shows:
- **Name & Description**: Your custom labels
- **Created Date**: When key was generated
- **Last Used**: Most recent activity
- **Status**: Active/Inactive toggle
- **Actions**: Edit, regenerate, or delete

#### Security Best Practices
- Use different keys for different environments
- Regularly rotate keys for security
- Monitor usage patterns for anomalies
- Revoke unused or compromised keys immediately

## Troubleshooting

### Common Issues

#### Dashboard Loading Problems
- Clear browser cache and cookies
- Disable browser extensions temporarily
- Check internet connection stability
- Try incognito/private browsing mode

#### Data Sync Issues
- Refresh the page to sync latest data
- Check for multiple browser tabs
- Verify API key permissions
- Contact support with browser console errors

#### Performance Optimization
- Close unused browser tabs
- Update to latest browser version
- Disable unnecessary browser extensions
- Use supported browsers (Chrome, Firefox, Safari, Edge)
