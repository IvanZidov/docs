## API Key Management

### Your Main API Key

Your primary key for all API access:

<Warning>
**Security First**: Never expose API keys in client-side code or public repositories.
</Warning>

#### Key Features
- **Copy to Clipboard**: One-click copying
- **Regenerate**: Create new key if compromised
- **Usage Tracking**: See when and where it's used
- **Rate Limits**: Current usage and limits

### Additional API Keys

Create specialized keys for different environments:

<Tabs>
  <Tab title="Development">
    Perfect for testing and development work.
  </Tab>
  <Tab title="Production">
    Your live application's secure key.
  </Tab>
  <Tab title="Staging">
    Testing environment before production.
  </Tab>
</Tabs>

#### Creating New Keys

<Steps>
  <Step title="Click 'Create New Key'">
    Start the key creation process.
  </Step>
  <Step title="Add Details">
    Give it a name and description (e.g., "Production App").
  </Step>
  <Step title="Copy Immediately">
    You'll only see the key once - copy it now!
  </Step>
  <Step title="Update Your App">
    Replace the key in your application code.
  </Step>
</Steps>
